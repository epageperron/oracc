Oracc XML Instance Set (XIS) API Documentation

Overview

Oracc is organized around the concept of the project, e.g., neo. Projects typically contain a corpus consisting of a catalogue, text editions, and one or more glossaries. The glossaries are named by ISO639 language codes, e.g., akk.

Glossaries are collections of entries, words or phrases, which have IDs of the form o12345678 or x12345678; these IDs are called OIDs (Oracc IDs). Work to make these OIDs persistent is under way but not yet complete.

Each entry contains several lists of instances of the words with particular features--the basic meaning, a finer-grained sense, various grammatical parts, etc. These instances are stored as lists whose abbreviation is XIS: XML Instance Sets, so we call the ID for such an instance sets an XIS

These elements and additional selection features make up the XIS API.
